Setup Guide
First-time setup
1. Get a free SubDL key at subdl.com
2. Get a free OpenSubtitles key at opensubtitles.com/consumers
3. Install Ollama: brew install ollama (macOS)
    or download from ollama.com (Windows/Linux)
4. Pull the translation model: ollama pull qwen2.5:7b
5. Start Ollama in a separate terminal: ollama serve
6. Enter your API keys above and click Save
